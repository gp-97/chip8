Detailed Description

Chip8 prototype class.

Contains all the data members and the associated member functions

The main components to be emulated include:

  • 16 bit Registers
  • 4K Memory
  • 16-bit Index Register
  • 16-bit Program Counter
  • 16-level Stack
  • 8-bit Stack Pointer
  • 8-bit Delay Timer
  • 8-bit Sound Timer
  • 16 Input Keys
  • 64x32 Monochrome Monitor
